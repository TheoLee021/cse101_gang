## Question 1

You are given two sequence of integers: $(A[1],\dots ,A[n]) \text{ and } (B[1],\dots ,B[m])$. You wish to find the length of the longest increasing common subsequence of the two lists (i.e., the length of the longest increasing subsequence $(A[i_{1}],\dots,A[i_{k}])$ such that there exists a subsequence $(B[j_{1}],\dots,B[j_{k}])$ such that $A[i_{1}]==B[j_{1}]$ and so on up to $A[i_{k}]==B[j_{k}]$. Design a DP tabulation algorithm that returns the length of longest common increasing sub sequence.

**1. Define the subproblem:**
Let $DP[i][j]$ be the length of the longest increasing common subsequence of the set $A[1],\dots,A[n] \text{ and } B[1],\dots,B[m]$ such that the common subsequence ends exactly with the element $B[j]$.

**2. Define and evaluate the base cases:**
The base for the either sequence is empty. Then $DP[i][0] = 0 \text{ for all } 0\leq i\leq n \text{ and } DP[0][j] = 0 \text{ for all } 0\leq j\leq m$.

**3. Establish the recurrence for the tabulation:**
- **Case 1**, if $A[i]\neq B[j]$. Since the subsequence ends must exactly with 
the element $B[j]$, the element $A[i]$ is not the last element of the subsequence. Then, $DP[i][j]=DP[i-1][j]$.
- **Case 2**, if $A[i] = B[j]$. This shows that the common increasing subsequence between $A \text{ and } B$ could be found. Then $DP[i][j]=1+Max(DP[i-1][k])$ where $B[k]<B[j]$ for all $1\leq k \lt j$.

**4. Determine the order of subproblems:**
Since we only calculate the previous values $DP[i-1][j]$ and $DP[i-1][k]$. The outer loop iterates through $i=1 \to n$. The inner loop iterate through $j=1\to m$.

**5. Final form of output:**
The final answer is $Max(DP[n][j])$ where $1\leq j\leq m$.

**6. Pseudocode:**

```java
LICS(A[n],B[m]):
	Init DP[i][0] = 0 for all i < n
	Init DP[0][j] = 0 for all i < m
	max_value = 0
	for index i from 1 to n:
		max_value = DP[i-1][0]
		for index j from 1 to m:
			if A[i] != B[j]:
				DP[i][j] = DP[i-1][j]
			else
				DP[i][j] = 1 + max_value
				
			if B[j] < A[i]
				max_value = max(max_value,DP[i-1][j])
	return max_value
```

**7. Runtime analysis:**
- Initial DP for $i \text{ and } j$ takes $O(n)+O(m)$.
- First iteration for $n$ takes $O(n)$.
	- Inner iteration for $m$ takes $O(m)$.
Then, the $LICS(A[n],B[m])$ algorithm takes $O(n)+O(m)+O(n\cdot m)$, which is considered as:
$$
O(n\cdot m)
$$
